# Admin Handbuch
> Dieses Handbuch richtet sich an Admins. Damit sind User mit der Rolle 'Admin' und nicht System-Admins gemeint.

## User Verwaltung

### User erstellen
Neue User können unter `domain.com/users` über den Knopf "User erstellen" angelegt werden. 
Nachdem Vorname, Nachname, Mail-Adresse sowie ein gültiges Passwort eingegeben und der Knopf "Erstellen" gedrückt wurde, ist der User sofort verfügbar.

### Mehr Aktionen
Unter dem '...' Knopf am Ende jeder User-Zeile verbergen sich noch weitere Aktionen:

#### Admin machen / entfernen
Damit ist es möglich einen User zum Admin zu machen / zu entfernen.

#### Passwort zurücksetzen
Sollte einmal ein Passwort verloren gehen, könnt ihr hier das Passwort direkt zurücksetzen.

#### Löschen
Ihr werdet in einem weiteren Dialog gefragt, ob ihr den gewählen User wirklich löschen wollt. Das muss bestätigt werden um den User zu löschen.
Um Admins zu löschen müsst ihr zuerst über den Knopf "Admin entfernen" den Account zu einem regulären User
machen, bevor ihr ihn löschen könnt.

## App Einstellungen

### Footer
Hier könnt ihr die Links konfigurieren, die den Usern im Footer auf der `domain.com` Seite angezeigt werden.
> Beachtet, dass es zwar eine grobe Validierung des URL Formats gibt, es jedoch in der Veranwortung der Admins ist,
> dass die Links auch wirklich stimmen
Der Link zu einer Kontakt-Seite und zu einem Impressum können angepasst werden, der Link zum Source Code kann nicht verändert werden.

## Tickets

### Tickets löschen
Nur Admins können Tickets löschen. Navigiert dafür zu `domain.com/tickets` und findet das Ticket, das ihr löschen wollt. Das könnt ihr dann auf 2 Wege tun:
- in der Ticket-Übersicht (`domain.com/tickets`) könnt ihr bei dem entsprechenden Ticket auf die 3 vertikalen Punkte klicken, um die weiteren Aktionen aufzurufen. Wählt dann das Mülleimer-Icon "Löschen" aus, um das Ticket zu löschen. Es öffnet sich ein Bestätigungs-Fenster. Überprüfe noch einmal, dass der Name des zu löschenden Tickets stimmt, und bestätige dann das Löschen über den roten "Bestätigen"-Knopf.
- ihr klickt auf das Ticket, das ihr löschen wollt, um in die Ticket-Detailansicht zu gelangen. Dort klickt ihr auf den roten Mülleimer-Knopf auf der rechten Seite, und bestätigt auch hier das Löschen über den roten "Bestätigen"-Knopf.

Solltet ihr ein Ticket erfolgreich gelöscht haben erscheint kurz eine Bestätigung am unteren Rand des Bildschirms.

## Labels

### Labels löschen
Nur Admins können Labels löschen. Navigiert dafür zu `domain.com/labels` und findet das Label, das ihr löschen wollt. Klickt auf den roten Mülleimer am Ende der Zeile des Labels, und bestätigt das Löschen  über den roten "Bestätigen"-Knopf.
Solltet ihr ein Label erfolgreich gelöscht haben erscheint kurz eine Bestätigung am unteren Rand des Bildschirms.
